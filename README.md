# IES_eDuca: Setup Guide

## Execution Guidelines

Ensure Docker is installed and operational on your system before proceeding.

---

### 1. Start Services

Run inside eDucaApp dir
```bash
./mvnw package
``

```bash
docker-compose up -d

```

### 2. Open the webApp
```bash
http://localhost:4200/

```
